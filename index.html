Pixel Rearrangement Painter

Load a source image and a target image. The tool remaps individual pixels from the source so they approximate the target's colors while smoothly animating the transition.

Tips
  • Higher resolutions yield better detail but require more computation.
  • You can drop new images at any time to recompute the pixel mapping with the current settings.
  • Use the progress slider while paused to scrub through the pixel motion.
Drop two images to begin.
Crafted with HTML, CSS, and vanilla JavaScript. No data leaves your browser.
